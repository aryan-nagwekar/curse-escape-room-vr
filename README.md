# Curse Escape Room VR

An interactive horror escape-room experience built with Three.js for a VR project competition.

The player is trapped inside a cursed ritual chamber hidden within an Egyptian pyramid. To escape,
they must solve environmental puzzles, recover the Skull of Morgath, the Amulet of Bael, and the
Tome of Shadows, place the relics into the altar molds, unlock the final passcode door, and walk out
into a moonlit Egyptian desert.

## Play

Open `index.html` or use the GitHub Pages URL once deployed.

## Included Assets

- `index.html`: complete standalone game
- `assets/skull_of_morgath.blend`: Blender asset for the skull relic
- `assets/rusty_chest_key.blend`: Blender asset for the chest key
